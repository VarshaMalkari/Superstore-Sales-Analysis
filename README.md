# Superstore Sales Performance Analysis (Python | SQL | Power BI)

## Project Overview
This project analyzes the Superstore dataset to identify key business insights related to **sales, profit, customer segments, regions, and discount impact**.  
The analysis was performed end-to-end using Python, SQL Server, and Power BI.

## Steps Covered
- Collected dataset from **Kaggle**
- Cleaned and prepared data using **Python (Pandas)**
- Performed exploratory visualizations using **Matplotlib**
- Imported cleaned data into **SQL Server (SSMS)** and executed business queries
- Built an interactive **Power BI dashboard** with KPIs and insights

## Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL Server (SSMS)
- Power BI
- GitHub

## Key Insights
- West region generated the highest profit
- Consumer segment contributed maximum profit
- Tables sub-category resulted in highest loss
- Texas was the worst performing state
- High discounts negatively impacted profitability
